Программа "Кот-декод" создана для автоматизированного шифрования и дешифрования данных пользователя, который авторизован в системе. 

Программа предоставляет пользователю следующие возможности: 
‒ полный доступ к программному обеспечению; 
‒ возможность ввода сообщения, его изменения или удаление данных, вывод зашифрованного и расшифрованного сообщений; 
‒ выдача документов в формате .txt
